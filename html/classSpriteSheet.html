Detailed Description

Class that handles the animation of the Player.

Using what is known as a spritesheet, this class handles the animation of the player dependent on the direction of said player. If the player is facing right for example, it loops through a series of 3 images in constant succession which gives the appearance of animated movement. Inherited from the animation class
