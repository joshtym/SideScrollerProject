Detailed Description

Class to handle all ObjectDimensions.

Assigns what is essentially a box around each object that uses this. Each object is assigned a minimum x and y value as well as a maximum x and y value as well as image height and width. Note that with allegro, minX and minY are the starting points of every image. Containers both setters and getters as well as an updater
